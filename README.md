# Glo+ APP

Glo+ React Native APP

## Description
App to help people to interact online while watching TV Globo.

## Running the app

### clone repository
```
git clone https://github.com/bacamartetech/gloplus-app
```

### change directory
```
cd gloplus-app
```

### install the dependencies
```
yarn // or npm install
```

### run on Android
```
yarn android
```

### run on iOS (not tested but should work)
```
yarn ios
```
